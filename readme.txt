# IBM-for-14_years_old Website

This is the readme file for the IBM-for-14_years_old website project.

## Description

The IBM-for-14_years_old website is a project aimed at creating a website with a similar design and functionality to the official IBM website but the goal is to make IBM understandable by 14 years 14 years old

## Installation

To run the IBM-like website locally, follow these steps:

1. Clone the repository: `git clone https://github.com/Nagrom850/IBM_3-mes`
2. Navigate to the project directory: `cd IBM_3-mes`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

## Usage

Once the development server is running, you can access the IBM-for-14_years_old Website by opening your web browser and navigating to `http://localhost:3000`.
